日记-小感慨

2021年07月07日 1.8k 字 大概 7 分钟

[日记] 眨眼间过去了一年

啪的一下很快啊!一下子过去了一年
感觉上来说,去年过的非常的曲折,心理层面上的曲折

前几天又想写东西,才想起来我已经搭建了一个博客
但是发送了挺多事情,又重装了一遍
现在用的Atom进行编写markdown,忘记了之前用的什么编辑器
目前用得挺好的，就是预览时无法实时展示图片

希望自己以后可以多多写博客,发现自己有这么一个小地方属于自己
可以无止境的去写一些内心感受真挺自在的
什么?正经人谁写日记放到这里?
去NM的,爱说啥说啥,爱写啥写啥,纯记录

回顾2020年

3月

在广州租新房子,养了一直猫,取名“多多”
全国新冠疫情爆发了!

4月

好不容易稳定下来
无任何学习与进步

5月

开始补习数学,打算开发MOD

然后后面服务器形式开始不太好转了….
6月

感觉这样下去不行,需要发展其他项目
开始研发MOD,市面上叫做xx引擎
回学校拿结业证有科挂了,干!

开始看数值策划的视频为了自己开服准备

7月|8月

除了开发引擎就是打王者
而且好像打王者的时间非常多且痴狂
而且做服务器技术的收入也慢慢没有了

有点温水煮青蛙的感觉..
9月|10月

还在开发引擎
在完全没有图形学的基础上硬磨图形学
还是实现了大部分功能的!
我比较开心,但是质量一般

在开发了2个月后发现期待的人不多
其实这太tm正常了,我觉得最主要原因是因为觉得自己写的不好
逐放弃了引擎开发,原本想靠引擎开发附属插件提高知名度
用知名度来打开市场,后续用附属插件收费
但同时期竞争产品较多,融合大部分产品难度较高
不融入个人感觉打开不了市场,也就是凭什么别人用你的?
但是我根本就没有投入市场,我畏惧别人的评价!

11月|12月

开始无头苍蝇式乱飞
不知道自己干什么要做什么即使现在写下这日记时也差不多//草
试了在虎牙开直播
由于长期睡眠不规律,所以凌晨直播
实验出无人观看时,人气值为2w8-3w左右
有1个人观看会达到4w人气
那时候播的是刺客信条英灵殿还tm有钱买游戏,你TM的

经过纠结
决定开设自己的服务器

2021年

1月

逐步开发基础插件
物品同步系统,装备进阶系统等等
个人感觉非常非常良好
过年时认识了一个玩家
她想赞助100来块
但是我拒绝了本次赞助,因为我觉得我的内容不够好
这是让我非常后悔的事情!
因为没有足够的资金就没有办法开发出更好的内容!
我因为害怕无法做得更好而不去接受这笔钱,我很后悔

2月

开发了很多的系统和完善了很多地方
数值策划的知识用到了吗？
要说用到也用到了,但是也没完全用到
只能说还不够!需要再多看多试验才可以做得更好

此时,虽然我的游戏功能开发得心里过得去
但是数值方面,美术方面,我觉得很不满意!
我愿意花更多的时间去打磨这个自己做的作品

但是我没有办法不拿出来,因为我缺钱
只能硬着头皮继续开发
这个月没有收入

3月

上个月月底时开发了强化系统
本月正式有了一些收入!
微信转账累计1k
月底制作了自助充值系统
合起来一个月赚了2k
我非常满意满足!
因为我以为我的内容没有办法让玩家付费
这一笔小钱给了我很大动力去更新

4月

这个月印象中没有去开发很多新的系统
一直在维护和调整新的数值
因为之前别人说数值幅度不够大
同时这里也出现了我个人设计的问题
因为我参考了造梦西游5游戏的设计
这款游戏的设计不适合用在多人游戏上!

5月

出现了越来越多的问题
同时因为我的直播来了更多的人
本月在收入上可以称得上是最佳了

于是我明白了一个道理!
别人进来充钱,可能就玩一下就走了
没有为什么,只需要把当时大家需要的服务给够就行!
一开始我以为大家会玩的很久,因为充值了之类的

后来非常焦虑和难受
因为我觉得我做的什么都不好,非常想推翻重做

6月

我取消了服务器的更新
我没有脸去群里通知
我没有脸去跟大家说话
我甚至不敢上QQ面对大家

我知道我内心不够强大,我不敢面对大家的指责不知道以后自己看到这段话什么感想哈哈!

##小小总结

现在我已经决定插本考试,努力上公办学校
我推翻了很多我自己以前的想法,幼稚的想法 谁不是在成长呢

关于服务器

希望以后自己要记住

  1. 不要再被玩家们左右了
    因为玩家群体很多,每个人想法不一样,一定要遵循自己的想法

  2. 要照顾充钱玩家的感受!
    不要做零氪金的体验!!!!为了钱不磕碜!!!

  3. 不要参考单机游戏的设计!一定要参考网络游戏
    不要去开发新玩法!目前市场已经非常稳定了
    赚钱且稳定的模式在市场上已经运行很久了!!

  4. 不要太过追求完美,完美主义会变成拖延症

我知道

  1. 服务器设计的不够好
  2. 幼稚导致了今天的局面
  3. 亏待了那些对我期待的玩家
  4. 贪玩,浪费时间,固执,自大,自满

未来

我希望

  1. 继续进修JAVA,很多以前知道不足的地方都要去补上!
  2. 学习数学,转而开发Unity3D
  3. 学习英语,因为我想看懂瓶子上的字!
  4. 如果可以,去看一些哲学!把它看完!目前在看耶鲁大学，死亡的哲学课,希望看完它

好了,想要记录的东西就以上这些了
我知道文字很TM不通顺,很TM难看
希望以后自己多多写博客
争取能提高自己的写作文化水平
啊？日记写多了应该也提高不了吧？
管他的，淦
